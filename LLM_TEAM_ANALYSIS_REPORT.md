# LLM 小组分析报告

## 组长

### 策略总体评估
回测结果显示策略在 2024 年白酒板块（BK0891）表现良好：
- **累计收益: +9.42%**，**年化收益: +67.11%** ✅
- **夏普比率: 2.05** ✅（非常优秀）
- **卡玛比率: 9.79** ✅（极佳）
- **胜率: 53.85%**，**盈亏比: 2.60** ✅

### 主要问题
1. **回测时间太短**：只有2024年11-12月的数据，覆盖面不够
2. **交易次数太少**：仅13次交易，平均持仓10.8天，策略选股能力有限
3. **入场评分阈值65太高**：导致很多机会被错过
4. **缺乏动态仓位管理**：固定15%仓位，没有根据市场/个股风险调整

### 改进方向
1. 降低入场门槛（50分），增加交易机会
2. 加入资金流向、板块动量等多维度信号
3. 增加择时模块，根据大盘走势调整仓位
4. 优化止损止盈参数

## 规划员

### 优势分析
1. 盈亏比2.60说明止盈止损设置合理
2. 夏普比率2.05，风险调整后收益优秀
3. 平均持仓10.8天，符合短线主线策略周期

### 劣势分析
1. 交易频率低（13次/年），资金利用率不足
2. 评分系统过于简单，仅用涨跌幅、成交额、股票代码前缀
3. 没有利用真实的主力资金流向数据

### 优化方案
1. **降低入场阈值**：65→45分，提高交易频率
2. **增加信号维度**：加入换手率、量比、MACD金叉等
3. **动态止盈**：采用移动止损，锁定更多利润
4. **加入择时**：根据大盘MA5/MA20金叉死叉调整仓位

#### 2. 增加信号维度评分
```python
def _score_stock(self, code, row, daily_data):
    score = 0
    # 涨跌幅 (0-30分)
    change_pct = float(row.get("change_pct", 0))
    if change_pct > 9: score += 30
    elif change_pct > 6: score += 25
    elif change_pct > 3: score += 20
    elif change_pct > 1: score += 15
    elif change_pct > 0: score += 10
    else: score += 0
    
    # 成交额 (0-20分)
    turnover = float(row.get("amount", 0))
    if turnover > 10e8: score += 20
    elif turnover > 5e8: score += 15
    elif turnover > 2e8: score += 10
    elif turnover > 1e8: score += 5
    
    # 换手率 (0-15分)
    turnover_ratio = float(row.get("turnover_ratio", 0))
    if turnover_ratio > 5: score += 15
    elif turnover_ratio > 3: score += 12
    elif turnover_ratio > 1.5: score += 8
    elif turnover_ratio > 0.8: score += 5
    
    # 股票板块加分 (0-10分)
    if code.startswith("60") or code.startswith("000"): score += 10
    elif code.startswith("688") or code.startswith("30"): score += 8
    
    return score
```

## 评审员

### 风险评估
1. **市场风险**：单一概念板块，高度集中，风险较大
2. **流动性风险**：成交额评分未考虑实际流动性
3. **模型风险**：评分系统过于简单，容易过拟合

### 改进建议
1. 增加止损触发后的冷却期（避免反复止损）
2. 加入大盘择时，降低系统性风险
3. 限制单只股票最大持仓比例
4. 增加止损次数统计，超过3次止损的股票加入黑名单
