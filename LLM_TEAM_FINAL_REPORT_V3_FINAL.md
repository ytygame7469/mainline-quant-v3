## 一、项目概述

**主线量化交易系统V3** 是一个专注于A股主线概念板块的量化交易系统，采用六层架构设计，融合传统量化策略与AI智能分析，目标实现稳定的市场超额收益。

### 核心设计理念

| 理念 | 说明 |
|------|------|
| 模块化架构 | 六层独立可替换，降低耦合，提高可维护性 |
| 数据驱动 | 多源数据融合，Parquet+DuckDB本地存储，支持离线分析 |
| 风控优先 | 四层风控体系，确保资金安全永远是第一位 |
| AI增强 | AI辅助而非替代，保留人工决策权 |
| 实盘就绪 | 完整交易闭环，从数据到执行一体化 |

## 三、LLM小组讨论决策

### 架构决策

- 六层架构: 数据引擎 → AI分析 → 策略引擎 → 风险控制 → 交易执行 → 监控通知
- 每层独立可替换，模块间通过接口通信

### 策略决策

| 策略 | 权重 | 核心逻辑 |
|------|------|----------|
| 主线轮动 | 45% | 12维度评分，识别并跟随主线 |
| 龙头战法 | 30% | 6维度识别真龙头，梯队管理 |
| 缠论择时 | 25% | 分型/笔/中枢，三类买卖点 |

主线生命周期: 萌芽 → 爆发 → 高潮 → 退潮

### 风控决策

| 模块 | 核心功能 |
|------|----------|
| 仓位管理 | 凯利公式、风险平价、单票/板块限额 |
| 止损止盈 | 固定止损(-8%)、ATR止损、移动止盈 |
| 风险预算 | 日亏损-2%、周亏损-5%、连续3笔熔断 |
| 压力测试 | 历史极端行情、蒙特卡洛模拟 |

### AI集成决策

- AI权重30% + 策略权重70%
- 4个专业Agent: 技术/基本面/资金/情绪分析师
- LLM选择: DeepSeek(性价比最优)

## 七、使用方式

### 安装

```bash
pip install -r requirements.txt
```

### 数据采集

```bash
python main.py collect --type daily --start 2024-01-01
```

### 主线扫描

```bash
python main.py scan --strategy mainline
```

### AI增强交易

```bash
python main.py trade --ai --capital 1000000
```

### 回测验证

```bash
python main.py backtest --strategy mainline --start 2024-01-01 --end 2024-12-31
```
